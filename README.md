# 🌿 植物圖鑑資料庫檢索系統

本專案為一個基於 **Spring Boot** + **Vue.js** 前後端分離架構開發的植物圖鑑檢索系統，支援 **即時動態模糊查詢功能**。使用者可透過搜尋引擎快速查詢植物資訊，包括名稱、分類、特徵與圖片等，並呈現直覺化的檢索體驗。

---

## 📦 技術棧

### 🔧 後端：Spring Boot
- Spring Boot 3.x
- Spring Data JPA / Hibernate
- MySQL
- RESTful API 設計
- CORS 支援
- 關鍵字模糊查詢

### 🎨 前端：Vue.js

---

## 🚀 功能特點

- 🔍 **植物搜尋引擎**
  - 支援英文植物名模糊比對

- 🌱 **植物詳情頁**
  - 顯示圖片、學名、俗名、科屬、特徵、用途等詳細資訊

- 📚 **資料管理（選配）**
  - 可儲存收藏喜愛的植
